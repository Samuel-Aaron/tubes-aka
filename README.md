# tubes-aka
Kelompok : The Struggler
Anggoota : Samuel Aaron - 103012580024 dan Marthin M Tambunan

Judul : Analisis Efesiensi Waktu Eksekusi Manajemen Inventori Barang Menggunakan Algoritma FIFO

Deskripsi Tugas : Studi ini berfokus pada optimasi manajemen inventori barang melalui Analisis Efisiensi Waktu Eksekusi pada sistem manajemen inventori menggunakan perbandingan algoritma iteratif dan rekursif. Fokus utama evaluasi terletak pada implementasi algoritma First-In-First-Out (FIFO) untuk mengukur performa pemrosesan data stok barang dalam skala besar.

Unduh dulu dependensi dalam virtual envrionment mu.
